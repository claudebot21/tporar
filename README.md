# TPOrar 🚌

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![React](https://img.shields.io/badge/React-18-blue)](https://reactjs.org/)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple)](https://getbootstrap.com/)

A simple React app to view Bucharest public transport routes and schedules. Shows buses, trams, and trolleybuses with quick links to official timetables.

## ✨ Features

- **Route Browser** — View all STB (Societatea de Transport București) lines
- **Organized by Type** — Separate sections for Trams, Trolleybuses, and Buses
- **Quick Schedule Links** — Direct links to official PDF schedules and route info
- **Multiple Operators** — Supports STB, STV, STCM, and RegioServ routes
- **Mobile Friendly** — Bootstrap-based responsive design

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/dekolor/tporar.git
cd tporar

# Install dependencies
npm install

# Start development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 🏗️ Build for Production

```bash
npm run build
```

Creates an optimized production build in the `build` folder.

## 🛠️ Tech Stack

- **React 18** — UI framework
- **React Bootstrap** — Component library
- **Axios** — HTTP client for API requests
- **STB API** — Real-time route data from info.stbsa.ro
